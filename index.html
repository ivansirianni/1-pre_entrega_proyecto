Primera Pre Entrega de Proyecto final

Esta app web esta dedicada a la industria farmaceutica. Se pensó armar una pequeña base de datos para una farmacia. En la app, el farmaceutico puede ver un catalogo, agregar productos nuevos que estan ordenados por id, tienen informacion de laboratorio, nombre del medicamento y precio. Se puede tambien eliminar uno o mas productos del catalogo. Pueden buscar productos por su ID o filtrar cuantos productos hay por laboratorios.
